ex) 이메일 등록 체크
- "가입하기" 버튼 클릭 인식
  DOM, JavaScript
- 이메일, 비밀번호 입력 여부 체크
  DOM, JavaScript
- 브라우저에서 이메일 등록 여부 체크 불가
- 입력한 값을 서버로 전송
  DOM, JavaScript
- 서버 프로그램에서 이메일 체크, 결과 전송
  JavaScript(node.js), Java 등
- 브라우저에서 결과 수신
  JavaScript
- 결과에 따른 처리
  HTML, CSS, DOM, JavaScript
  
### 문장
- JS 코드 실행단위
- 세미콜론(;)까지 하나의 문장
- var book = "책";

```
if (true) {
  var book = "책";
  var point = 123;
};

if (true) {
  var book = "책";
};
```

- 문장 시작 위치
  - 위치 제약 없음
  - 들여쓰기: 일반적으로 2칸, 4칸 (개발자 취향대로)
